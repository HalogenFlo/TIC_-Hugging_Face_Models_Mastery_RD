🚀 Nâng Cấp Hệ Thống & Tính Năng Mới

Nhật ký cập nhật chi tiết các tính năng, sửa lỗi UI và cơ chế xuất file PDF mới nhất

Phiên bản 2.5.0

1. Tính Năng Xuất Báo Cáo PDF Mở Tab Mới (`target="_blank"`)

Cơ chế hoạt động: Khi người dùng yêu cầu "xuất báo cáo" hoặc "tải pdf", hệ thống tự động biên soạn bài tư vấn đầy đủ và sinh file báo cáo HTML/PDF lưu vào app/exports/.

  • Mở Tab mới (Không chuyển hướng Chat): Liên kết tải file gán thuộc tính target="_blank" giúp trang chat hiện tại giữ nguyên 100%.
  • Mẫu Báo Cáo Chuẩn A4 Văn Phòng: Tích hợp phông chữ Segoe UI / Roboto, Tiêu đề Quốc hiệu trang trọng, và thanh công cụ bật tự động cửa sổ window.print() (Save as PDF).
  • Sửa lỗi vỡ định dạng Markdown: Tích hợp bộ parser convert_markdown_to_html tự động biến các Bảng biểu Markdown (như biểu thuế lũy tiến) thành bảng HTML .legal-table kẻ viền chuẩn mực.

2. Chiến Lược Phân Loại Trả Lời & Trích Dẫn Chuẩn

  • Mặc định Ngắn Gọn: Trả lời cực kỳ ngắn gọn, đi thẳng vào đáp án trọng tâm khi hỏi thông thường.
  • Full Report Mode: Trình bày bài tư vấn đầy đủ, chuyên nghiệp, đẹp mắt như ChatGPT khi có yêu cầu "xuất báo cáo".
  • Quy tắc Thời gian: Mặc định 100% áp dụng Luật Mới Nhất Hiện Hành nếu người dùng không nêu mốc thời gian.
  • Định dạng Trích dẫn Chuẩn: Cấu trúc [Số hiệu văn bản - Điều X - Khoản Y - Điểm Z].

3. Bộ Nhớ Hồ Sơ Khách Hàng Dài Hạn (User Profile)

Lưu giữ globalUserProfile vĩnh viễn trong localStorage và CSDL SQLite tic_personal_memory.db. Bấm "+ Cuộc hội thoại mới" vẫn bảo tồn 100% dữ kiện hồ sơ khách hàng tích lũy.
